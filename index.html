Core

alerts

The following alerts will be use throughout both implementations. There are 4 different styles available,

  • Warning, notify the user if something is not working correctly
  • Error, An element had broken and had produced an error
  • Success, A process has completed successfully
  • Info, Provide the user with extra information

Example

Warning Message! Lorem ipsum dolor sit amet, consectetur adipisicing elit.
Error Message! Numquam quos fuga quam suscipit sapiente perferendis magnam.
Success Message! Totam officiis dolorum voluptatibus maxime molestiae iste.
Info Message! Consequatur facere deleniti cumque ducimus maiores nemo.

Map Browser

Maps Implementation

This is how the Google Maps api will be implementation. The standard Google Maps interface will be disabled to focus of the custom pins added to the map.

Custom Pins

  • The custom pins will be formatted with a custom icon that is relevant to the point of interest.
  • The longitude and latitude of each pin is to be added to the code

Information Windows

  • Each pin will have a popup window with information on the location.
  • The content is formatted in the same styles as the rest of the page.

Example

Back to Top

    <script>
      var map;
      function initMap() {
        map = new google.maps.Map(document.getElementById('map'), {
          zoom: 14,
          center: new google.maps.LatLng(-34.928963, 138.600711),
          mapTypeId: 'roadmap'
        });

        var iconBase = 'https://maps.google.com/mapfiles/kml/shapes/';
        var infoicon = iconBase + 'info-i_maps.png';

        var contentString = '<div id="content">'+
            '<h1 class="firstHeading">Adelaide CBD</h1>'+
            '<div id="bodyContent">'+
            '<p>Adelaide is South Australia’s cosmopolitan coastal capital. Its ring of '+
            'parkland on the River Torrens is home to renowned museums such as the '+
            'Art Gallery of South Australia, displaying expansive collections including '+
            'noted Indigenous art, and the South Australian Museum, devoted to natural '+
            'history. The citys Adelaide Festival is an annual international arts '+
            'gathering with spin-offs including fringe and film events.</p>'+
            '</div>'+
            '</div>';

            var infowindow = new google.maps.InfoWindow({
              content: contentString
            });

        // Create markers.
          var marker = new google.maps.Marker({
            position: new google.maps.LatLng(-34.928963, 138.600711),
            icon: infoicon,
            map: map
          });
          marker.addListener('click', function() {
            infowindow.open(map, marker);
          });

      }
    </script>
